.. _spray-io:

spray-io
========

*spray-io* provides a low-level network IO layer for directly connecting Akka actors to asynchronous Java NIO sockets.
We like to think of it a basic version of Netty_ for Scala.

.. _netty: http://www.jboss.org/netty
