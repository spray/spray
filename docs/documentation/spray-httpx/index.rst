.. _spray-httpx:

spray-httpx
=============

The *spray-httpx* module contains all higher-level logic for working with HTTP messages, which is not specific to either
the server-side (:ref:`spray-routing`) or client-side (:ref:`spray-client`) modules on top and therefore (potentially)
used by both of them.
